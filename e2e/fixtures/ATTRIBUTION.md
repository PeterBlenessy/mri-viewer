## About Cornerstone

Cornerstone is a complete web based medical imaging platform that provides:
- DICOM image rendering
- Medical image processing
- Image manipulation tools

The Cornerstone project is maintained by the Open Health Imaging Foundation (OHIF).

## Files Included

The following test DICOM files are used for E2E testing:

- `single-image.dcm` - CT test image from Cornerstone test suite
- `multi-series/*.dcm` - Additional test images for navigation testing

These files are:
- ✅ Already anonymized (no patient data)
- ✅ Publicly available
- ✅ Licensed under MIT (compatible with this project)
